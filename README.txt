Clone the repo
**Follow these commands**
1.git pull origin main
2.git checkout -b your-branch-name
3.Implement your changes or new features in this branch.
4.After completion of your work
5.git add .
6.git commit -m "commit-msg"
7.git push origin your-branch-name
8.On GitHub, navigate to the repository.
9.Click on the "Pull requests" tab.
10.Click "New pull request."
11.Select your branch from the compare dropdown.
12.Review the changes and click "Create pull request."
13.Provide a clear title and description for your PR.
